## What Is This?

**Trade Launch Intelligence** is a Claude AI skill that functions as a full advisory team for trade and export businesses. In a single session, it replaces:

| Professional | What They Do | This Skill Covers |
|---|---|---|
| Market Research Analyst | Market sizing, trends, competitive intelligence | ✅ Phases 1–2 |
| Business Strategist | SWOT, strategy options, go-to-market | ✅ Phase 2 + DOC 2 |
| Financial Analyst | Financial model, ROI, projections | ✅ Phase 7 + DOC 3 |
| Regulatory/Compliance Advisor | Licensing, certifications, export law | ✅ Phases 5–6 + DOC 4 |
| Supply Chain Specialist | Sourcing, logistics, quality control | ✅ Phase 3 + DOC 5 |
| Risk Advisor | Risk register, mitigation, contingency | ✅ Phase 8 + DOC 6 |

A consulting firm would charge **$50,000–$500,000** for this package. This skill delivers it in minutes — for any product, in any country.

## Quick Start

### Installation

**Method 1: Direct Install (Recommended)**
1. Download the latest `.skill` file from [Releases](https://github.com/yourusername/trade-launch-intelligence/releases)
2. Open [Claude.ai](https://claude.ai)
3. Go to **Settings → Skills → Install Skill**
4. Upload the `.skill` file
5. Start a new conversation

**Method 2: Build from Source**
```bash
git clone https://github.com/yourusername/trade-launch-intelligence.git
cd trade-launch-intelligence
bash install.sh
```

### First Use

Once installed, trigger the skill with any of these phrases:

```
"I want to start exporting [product] from [country]"
"Is there money in [product]?"
"Help me research a product to trade"
"What should I export from [country]?"
"I have $5,000 and want to start a trading business"
"Give me a business plan for exporting cashew nuts"
"Write me a pitch email for my shea butter export business"
```
